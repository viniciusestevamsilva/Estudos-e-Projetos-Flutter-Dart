# Atividade 1 Desenvolvimmento Movel

<br>

## 📌 Observação

Fazer todas com NULL SAFETY e outras validações, TODAS com ENTRADA DE DADOS

<br>

## 📍 Súmario

<br>

```

Desenvolvimento Móvel: Atividade 01 (Tentem produzir códigos com entrada de dados e validações com IF)

[✔️] ➖ Faça um programa que mostre a mensagem "Olá Mundo!" na tela.

[✔️] ➖ Faça um programa que peça 2 valores. Calcule e imprima  a soma, o produto, a subtração, a divisão, o resto da divisão e a divisão inteira.

[✔️] ➖ Faça um programa que peça um ano qualquer. O programa deverá calcular e imprimir a idade.

[✔️] ➖ Faça um programa que peça 4 notas com  entrada de dados. O programa deverá calcular a média das notas digitadas.

[✔️] ➖ Faça um programa que receba e divida 2 números. A saída da divisão precisará ser formatada com 4 casas decimais.

[✔️] ➖ Faça um programa que recebe um número inteiro e mostre o sucessor e antecessor.

[✔️] ➖ Faça um programa que receba um número qualquer e calcule o dobro e o triplo desse número.

[✔️] ➖ Faça um programa para converter medidas

[✔️] ➖ Faça um programa que receba um número inteiro. O programa deverá imprimir tabuada de multiplicação deste número.

[✔️] ➖ Faça um programa que receba um valor em reais, depois calcule quantos euros, dólares e wons dariam para comprar com esse valor.

[✔️] ➖ Faça um programa com entrada de dados para calcular a área de 4 objetos.

[✔️] ➖ Faça um programa para receber um número qualquer e informar na tela se é par ou ímpar. 

[✔️] ➖ Faça um algoritmo que leia dois valores inteiros A e B. Se os valores forem iguais deverá somar os dois, caso contrário multiplique A por B. Ao final de qualquer um dos cálculos deve-se atribuir o resultado para uma variável C e mostrar seu conteúdo na tela. 

[✔️] ➖ Faça um programa para encontrar o dobro de um número caso ele seja positivo e o seu triplo caso seja negativo, imprimindo o resultado. 

[✔️] ➖ Faça um programa que leia uma variável e some 5 caso seja par ou some 8 caso seja ímpar, imprimir o resultado desta operação. 

[✔️] ➖ Faça um programa que leia três valores inteiros e diferentes e mostre-os em ordem decrescente.
 
[✔️] ➖ Faça um programa para calcular o IMC COMPLETO de uma pessoa.

[✔️] ➖ Faça um programa para conversão de temperaturas (C => F  e F ⇒ C)

[✔️] ➖ Faça um programa para calcular juros.

```
